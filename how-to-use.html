How to Use

Welcome to the Cosmic Text Template Replacer! Follow these steps to use this tool:

Step 1: Enter a template text in the "Enter Your Text" field. Use $k as a placeholder wherever you want values to be replaced.

Step 2: In the "Enter Values" field, type values separated by commas. Each line should represent a new set of values for the placeholders in the template.

Step 3: Click "Generate" to see the filled text with placeholders replaced by the values provided. You can also use "Copy Output" to copy the generated text to your clipboard.

Example:

Template: "My name is $k and I live in $k"

Values: "Ramesh, Bangalore"

Output: "My name is Ramesh and I live in Bangalore"
